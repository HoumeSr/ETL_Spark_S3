# ETL с Spark и Minio
В первую очередь требуется скачать модули для работы Spark и Minio
```bash
make jars-download
```
Чтобы поднять инфаструктуру можно использовать:
```bash
make up
```
Одной из команд ниже можно сгенерировать данные для таблиц order, store и user

```bash
make generate
```

Они должны находиться в директории  
```s3a://warehouse/parquet/```

Чтобы получить таблицу результата, требуется написать
```bash
make get_result
```
Он будет находится в той же директории, что и исходные данные.
